Demo set of 175 users

This is a synthetic set of 175 Twitter users that have been manually determined to have a single dominant interest that they post about consistently. Although data will never be this clean in practical use (users may have multiple or no identifiable interests), such a ground truth allows us to evaluate the effectiveness of the method compared to others, and serves as a clean and predictable demo to showcase the capability of the system.
